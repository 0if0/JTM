# Security

This repository contains **Jenkins Test Management (JTM)**, a Jenkins plugin (`jtm-test-management`). Security vulnerabilities in Jenkins and plugins published by the Jenkins project are handled according to the [Jenkins security process](https://www.jenkins.io/security/).

## How to report

**Please report security issues in the Jenkins issue tracker under the [SECURITY project](https://issues.jenkins.io/secure/CreateIssueDetails!init.jspa?pid=10180&issuetype=10103).** That project is restricted so that only the reporter, maintainers, and the Jenkins security team can see the details.

Guidelines and the full workflow are documented in [Reporting security vulnerabilities](https://www.jenkins.io/security/reporting/).

If you cannot use the issue tracker, send your report to the private Jenkins Security Team mailing list: **jenkinsci-cert@googlegroups.com** (the team can file an issue on your behalf).

## Please do not

- Open a **public** GitHub issue (or similar) for an undisclosed security vulnerability; use the SECURITY project or the mailing list above instead.
